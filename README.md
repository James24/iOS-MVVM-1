# iOS Model-View-ViewModel

## Startup

  0) Install Bundler

  `gem install bundler` 

  1) Install Cocoapods

  `bundler install`

  2) Install Pods
  
  `pod install`

  3) Open Workspace
  `open iOS-MVVM.xcworkspace`
